# ng-sandbox
Angular Sandbox for experimentation

```
npm i -g @angular/cli
npm install -g json-server
```

```
cd demo-app
ng serve --open
```

```
json-server --watch mock/db.json
```

https://www.gerome.dev/blog/standalone-angular-folder-structure/
